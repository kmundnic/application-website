About

Hi! My name is Karel Mundnich, and I'm an Electrical Engineer. This website is a place to store code snippets, ideas, project descriptions, and part of my photography portfolio.

I'm currently interested in the use of signal processing and machine learning tools in order to understand the behavior of people at work. I believe that this can be a very powerful tool to understand working habits, and get insights about productivity, procrastination, and health.
