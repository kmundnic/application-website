Pelican lets you include images via Markdown,

and code snippets with a specific syntax. This is an example for Python,

print "Hello World!"

while this is an example for Matlab:

disp("Hello World!");

Finally, C's "Hello World!"

/* Hello World program */

#include<stdio.h>

main() {
    printf("Hello World");
}
