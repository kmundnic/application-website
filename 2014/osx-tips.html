OS X Tips

Tue 09 December 2014 | tags: OS X, Terminal, -- (permalink)

Launch software from Terminal.app:

  • SourceTree
  • Sublime

Command line tools (I always forget these)

grep -rwn 'directory' - "pattern"

Block webpages (e.g., Facebook):

In Terminal.app,

sudo nano /etc/hosts

Add a line with:

127.0.0.1 www.facebook.com
