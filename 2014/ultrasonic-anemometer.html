Ultrasonic Anemometer

Wed 15 January 2014 | tags: project-portfolio, -- (permalink)

  • Project title: Open-source Ultrasonic Anemometer
  • Code: Repository on Google Code

Summary

This project started as an initiative to create a cheap and open-source ultrasonic anemometer, in order to create and install networks of anemometers that may help meteorologists better understand the physics of the atmosphere in complex scenarios, such as mountainous areas.

The anemometer was design to be connected to a computer through a USB port. Unlike commercial solutions, where special electronics are created to calculate the speed of sound, the data is retrieved and analyzed in a computer, giving the possibility of using higher-level languages such as Python for data analysis.

Work

In this project, I worked as a software engineer, writing the code that calculates the speed of the wind in two axes, by measuring the speed of sound. Specific tasks included:

  1. Research the state-of-the-art in signal processing to calculate time of flight of ultrasonic waves.
  2. Implement some of the current techniques to calculate the time of flight.
  3. Create new methods to calculate the time of flight, based on our own data, which included non-linear noise, and references away from 0[V].
  4. The algorithms were implemented in Python, using Scipy and Numpy.
  5. Testing the anemometer in wind tunnels, using standard measurement equipment
  6. Writing documentation for future developers.
